### 问答作业

1. SV39 页表项由 Reserved 字段、物理页号和标志位组成。标志位可以避免用户态访问内核空间，以及避免不合法的内存访问、避免误修改重要程序和数据。
2.  + `Load access fault` `Store access fault` `Instruction page fault` `Load page fault` `Store page fault`
    + 发生缺页时，`scause` 为错误原因，`stval` 为尝试访问的错误虚拟地址，`sepc` 为发生缺页时的指令地址。
    + Lazy 策略的好处是可以避免占用不会立即使用的物理内存，同时减少不必要的磁盘 I/O 开销。
    + 约 20M。
    + 在分配大内存时，若分配的区域完全包含了当前层级页表的地址空间，则不立即向下继续分配下一级页表，而是直接将分配内存的权限位记录在当前的高层级页表。当发生缺页时，OS 可以检查含有权限位的高层级页表，从而用合适的权限位新建下一级页表。
    + 此时 PTE 的 V 标志位被置为 0，其它位记录被换到磁盘上的页面所处的位置。
3.  + 当进程切换时，直接将新进程的页表 token 写入 `satp` 寄存器即可。
    + 在内核页面上将 U 标志位置为 0。
    + 单页表可以使得内核在处理系统调用时，不需要手动遍历用户态的页表取得用户态虚拟地址所对应的物理地址，而可以直接通过虚拟地址进行访问。
    + 双页表下在系统调用和进程切换时需要更换页表；单页表下只需要在进程切换的时候更换页表。
